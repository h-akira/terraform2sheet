Terraform2Sheet 出力サンプル（通常表示）

例1: 単純な属性のみ（aws_example_resource.simple）

パラメータ 値 必須 デフォルト 説明
aaa value_aaa Yes - AAAパラメータの説明
bbb value_bbb No default_bbb BBBパラメータの説明
ccc 123 No - CCCパラメータの説明（数値）

例2: 配列属性（aws_example_resource.array）

パラメータ 値 必須 デフォルト 説明
name example_name Yes - リソース名
tags 1 tag_value_1 No - タグの配列
2 tag_value_2 No - タグの配列
3 tag_value_3 No - タグの配列

例3: ネストされた配列（aws_example_resource.nested）

パラメータ 値 必須 デフォルト 説明
name nested_example Yes - リソース名
rules 1 name - rule_1 No - ルールの配列
methods 1 GET No - 許可されるHTTPメソッド
2 POST No - 許可されるHTTPメソッド
3 PUT No - 許可されるHTTPメソッド
priority - 100 No - ルールの優先度
2 name - rule_2 No - ルールの配列
methods 1 DELETE No - 許可されるHTTPメソッド
priority - 200 No - ルールの優先度
enabled true No false リソースの有効/無効

例4: 参照とペンディング値（aws_example_resource.references）

パラメータ 値 必須 デフォルト 説明
name example_with_refs Yes - リソース名
computed_id null - (computed) Apply後に決定される値
attached_items 1 arn:aws:service::123456789012:resource/existing-item - - アタッチされたアイテムのARN
2 (pending) aws_example_item.item_a - - アタッチされたアイテムのARN
3 (pending) aws_example_item.item_b - - アタッチされたアイテムのARN
