Test003 - Irregular Nesting Depths

Generic JSON Structure (Non-AWS)

This test verifies that the table generator handles:

  • Simple values (string, number, bool)
  • Shallow arrays
  • Very deep nesting (5 levels)
  • Mixed depths in the same structure
パラメータ 値 必須 説明
simple_string hello Yes シンプルな文字列
simple_number 42 No シンプルな数値
simple_bool True No シンプルな真偽値
shallow_array 1 a No 浅い配列
2 b No 浅い配列
3 c No 浅い配列
deep_nested - level1 - level2 - level3 1 items 1 name - item1 No 深くネストされたオブジェクト（5階層）
value - 100 No 深くネストされたオブジェクト（5階層）
2 name - item2 No 深くネストされたオブジェクト（5階層）
value - 200 No 深くネストされたオブジェクト（5階層）
2 items 1 name - item3 No 深くネストされたオブジェクト（5階層）
value - 300 No 深くネストされたオブジェクト（5階層）
another_simple world - もう一つのシンプルな値
mixed_depth 1 shallow_field - x No 浅いフィールドと深いフィールドが混在
deep_field - nested - value - y No 浅いフィールドと深いフィールドが混在
